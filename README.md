#Spotify Clone

A Spotify clone created using HTML and CSS. This project demonstrates the use of various web technologies and techniques to replicate the look and feel of the Spotify web interface.

--------Features---------
Responsive design that adapts to different screen sizes
Custom range inputs using WebKit for enhanced control and customization
Clean and modern UI inspired by Spotify

----------Technologies Used-----------
HTML: For the structure of the web pages
CSS: For styling and layout
WebKit: For customizing input range elements

----------Usage--------
To view the Spotify clone, simply open the index.html file in your web browser. The design is fully responsive and should work well on both desktop and mobile devices.

---------Concepts Applied-----------
WebKit for Input Range: Utilized WebKit-specific properties to control and customize the appearance of range inputs. This allows for fine-tuned styling and improved user experience.
